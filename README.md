# 🎭 Дуплед (Dupled) — Tinder для творческих коллабораций

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Status: Concept / MVP](https://img.shields.io/badge/Status-Concept-blue.svg)]()

**Дуплед** — это WEB-приложение, созданное для того, чтобы творческие люди находили друг друга не для романтических свиданий, а для создания крутых проектов.

Забудьте про бесконечные чаты в Telegram «ищу команду». Здесь художник находит поэта для зина, музыкант — видеографа для клипа, а backend-разработчик — дизайнера для стартапа. Свайпай вправо — если хочешь в команду, влево — если не по пути.

---

## 🚀 Суть проекта (Elevator Pitch)

**Проблема:** Творческие люди часто варятся в собственном соку или тратят недели на поиск "того самого" специалиста в городских пабликах.

**Решение:** Механика Tinder/Bumble, адаптированная под деловые и творческие связи.  
1. Указываешь, **кто ты** (Художник, Копирайтер, 3D-моделлер).  
2. Указываешь, **кто тебе нужен** (Звукорежиссер, Python-разработчик).  
3. Алгоритм показывает тебе людей, чьи навыки совпадают с твоим запросом, и наоборот.
